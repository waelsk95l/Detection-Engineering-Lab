# Phase 1 – PowerShell Detection

This phase focuses on detecting PowerShell-based attacks using Windows Event Logs.

Cases included:

- Case1 PowerShell Detection
- Case2 Encoded PowerShell
- Case3 Suspicious Parent Process
- Case4 PowerShell Rundll32
- Case5 Hidden PowerShell
- Case6 PowerShell Download Payload
- Case7 PowerShell Download Cradle
- Case8 Encoded PowerShell Download
